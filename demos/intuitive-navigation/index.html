ScrollIntent.js

Intuitive Navigation Demo

In this demo, scrollIntent.js is used to intuitively hide/show the header when it appears the user is focused on content/navigating elsewhere.

ScrollIntent.js is given 5 actions that describe the behaviour of a user and what functionality should be executed based on that behaviour.

Actions

The following actions are used to hide/show the navigation header when necessary:

  • Hide the header when the user scrolls down past a waypoint at 300px
  • Show the header when the user scrolls up past a waypoint at 300px
  • Show the header when a user scrolls faster than 60px per 100ms, anywhere after the waypoint at 300px
  • Hide the header when a user scrolls slower than 60px per 100ms, for at least 500ms, anywhere after the waypoint at 300px (using only the scrollbar or mousewheel)
  • Hide the header when a user scrolls slower than 60px per 100ms, anywhere after the waypoint at 300px (using only the keyboard's up/down keys)

In code, these actions are written like so:

var actions = [
  {
    waypoint: 300,
    direction: "down",
    callback: function() {

      preventShow = true;
      menu.hide();
    }
  },

  {
    waypoint: 300,
    direction: "up",
    callback: function() {

      menu.show();
    }
  },

  {
    minWaypoint: 300,
    minSpeed: 60,
    callbacksPerAction: 1,
    callback: function() {

      preventHide = true;

      if(preventShow === false) {
        menu.show();
      }
    }
  },

  {
    minWaypoint: 300,
    maxSpeed: 60,
    scrollMethod: ["scrollbar", "mousewheel"],
    minDuration: 500,
    callbacksPerAction: 1,
    callback: function() {

      if(preventHide === false) {
        menu.hide();
      }
    }
  },

  {
    minWaypoint: 300,
    maxSpeed: 60,
    scrollMethod: ["down", "up"],
    callbacksPerAction: 1,
    callback: function() {

      if(preventHide === false) {
        menu.hide();
      }
    }
  }
];

// Setup scrollIntent
var scrollIntent = new ScrollIntent(window, actions, options);


scrollIntent.scrollingStopped = function() {
  preventShow = false;
  preventHide = false;
}

Try scrolling past the waypoint/at difference speeds to see the header hide/show as necessary.
